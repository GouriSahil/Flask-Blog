About Flask Blog

A focused blogging demo built with Flask—styled with your Vanilla Latte and Teal Forest palette for a calm, professional reading experience.

Project status

This project is in active learning and demo mode. Some demo copy may appear as placeholders; core features—accounts, posts, uploads, and password reset—are wired for real use.

Overview

Flask Blog shows how to structure a small Flask application with blueprints, SQLAlchemy models, authentication, and secure forms—without sacrificing a polished front end.

Key features

  • Registration & login
  • Create, edit & delete posts
  • Profile images
  • Password reset flow
  • Responsive Tailwind UI
  • CSRF-aware forms

Technology stack

Backend

  • Python & Flask
  • SQLAlchemy
  • Flask-Login & WTForms

Frontend

  • Tailwind CSS
  • PostCSS pipeline
  • Font Awesome icons
